⚠️ Development Mode

This is a static HTML demonstration. For full functionality:

  • Install dependencies: npm install --legacy-peer-deps
  • Set up PostgreSQL database
  • Run Prisma migrations: npx prisma generate and npx prisma migrate dev
  • Start development server: npm run dev
